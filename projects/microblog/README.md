此项目使用 Python 虚拟环境 virtualenv 来构建 Python 运行环境

.gitignore 忽略此项目下的 flask 目录

如果同步后不存在 flask 目录，则手动执行以下步骤

1. 安装 virtualenv
2. 在 ./flask/Scripts/ 目录下安装 flask 包及相关包

'pip install virtualenv'

安装包
- flask
- flask-login
- flask-openid
- flask-mail
- flask-sqlalchemy
- sqlalchemy-migrate
- flask-whooshalchemy
- flask-wtf
- flask-babel
- guess_language
- flipflop
- coverage
